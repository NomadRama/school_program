Raspberry Pi Pico

Микроконтроллер RP2040, 40 пинов, USB Micro-B. Компонент нейтральный — подсветка активных пинов добавляется в каждой схеме отдельно.

Типы пинов

ТипПримерыОписание
gpioGP0…GP22Цифровые входы/выходы, 3.3V
adcGP26, GP27, GP28Аналоговые входы + цифровые
powerVBUS, VSYS, 3V3Линии питания
groundGND, AGNDЗемля (8 пинов всего)
controlRUN, 3V3_ENУправление режимом платы
refVREFОпорное напряжение ADC
