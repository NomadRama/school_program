3-модуль — Режимдері бар түнгі шам

Біркелкі жарықтық, бірнеше режим, түйме арқылы ауысу. 1- және 2-модульдерден кейін ашылады.

Не күтеді сізді

Осы модульде сіз үйренесіз:

  • LED жарықтығын біркелкі басқаруды (тек «қосу/өшіру» емес)
  • ИМҰ (PWM) — Pico пиндерінің арнайы режимін пайдалануды
  • Pico кірісіне түйме басылуын оқуды
  • Бірнеше режимді ауыстыруды: жарық / күңгірт / жыпылықтау / «тыныс алу»

Модульде жаңа: аналогтық басқару (ИМҰ), кіріс ретінде түйме, күйлер.

Алдын ала не білу керек

3-модуль — 1-кезеңнің аяқталуы. Алдымен 1- және 2-модульдерден өтіңіз — осында бәрі қажет болады: Pico, түйме, реле, if/else шарттары, ЖИ-серіктес.
