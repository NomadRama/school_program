Модуль 3 — Ночник с режимами

Плавная яркость, несколько режимов, переключение кнопкой. Открывается после Модулей 1 и 2.

Что вас ждёт

В этом модуле вы научитесь:

  • Управлять яркостью LED плавно (а не только «включил/выключил»)
  • Пользоваться ШИМ (PWM) — специальным режимом пинов Pico
  • Считывать нажатие кнопки на вход Pico
  • Переключать несколько режимов: ярко / тускло / мигание / «дыхание»

Новое в модуле: аналоговое управление (ШИМ), кнопка как вход, состояния.

Что нужно знать заранее

Модуль 3 — завершение Этапа 1. Сначала пройдите Модули 1 и 2 — здесь пригодится всё: Pico, кнопка, реле, условия if/else, ИИ-напарник.
