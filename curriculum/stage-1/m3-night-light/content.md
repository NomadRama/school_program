# Модуль 3 — Ночник с режимами

> **Статус: черновик.** Разрабатывается после Модулей 1 и 2.

LED с плавной яркостью, несколько режимов, переключение кнопкой. Первый проект с **ШИМ (PWM)** — аналоговым управлением.

---

## Главное новое

- **ШИМ (PWM)** — плавное управление яркостью, не только «вкл/выкл»
- **Кнопка как вход Pico** — ученик считывает нажатие из кода
- **Состояния устройства** — несколько режимов, переключаются кнопкой
- **Прямое подключение LED к Pico** — без реле (сигнальный LED, маленький ток)

## Железо (предварительно)

- Raspberry Pi Pico
- Плата расширения ATX
- Светодиод (обычный 5мм, с резистором 330Ом)
- Тактовая кнопка
- Провода Dupont
- **Реле в M3 не используется** — направление упрощения

## Спираль — что повторяется из M1 и M2

- Основы цепи, замыкание, GND, мультиметр (из M1)
- Мастер-промпт ИИ (общий паттерн, обновлённый под ночник)
- Pico + GPIO (общее)
- `while True` + `sleep` (из M1), `if/else` (из M2)

**Новое:** PWM, считывание кнопки с Pin.IN, state machine простая.

## Режимы (точки выбора)

Минимум 3 режима, ученик добавляет свой:
- ☀️ Ярко — 100% яркости
- 🌅 Тускло — 20% яркости
- 🌙 Дыхание — плавное увеличение и уменьшение
- *Свой вариант* — например, имитация свечи, мигание азбукой Морзе

## План разработки

Создаётся после пилотного запуска M1+M2.
