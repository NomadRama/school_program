# Этап 1 — База

Первый этап программы ShokanLab. **Вводит ученика в инженерию через 3 спиральных модуля** — каждый проект повторяет основы предыдущего и добавляет одно новое понятие.

## Длительность

До 18 занятий (по 6 на модуль), свободный темп. На практике ученики проходят за 3-5 месяцев.

## Модули

| № | Модуль | Главное новое | Статус |
|---|---|---|---|
| 🟡 | [intro — Введение](intro/) | знакомство со школой, безопасность, ИИ | готов |
| 💡 | [m1 — Управляемая лампа](m1-controllable-lamp/) | цепь, кнопка, реле, первый код, мастер-промпт ИИ | готов |
| 💨 | [m2 — Умная сушилка](m2-smart-dryer/) | датчик как вход, `if/else`, 12V нагрузка | черновик |
| 🌙 | [m3 — Ночник с режимами](m3-night-light/) | ШИМ (PWM), прямое подключение LED | черновик |

## Спираль (Bruner)

Модули 1-3 **намеренно похожи по структуре**. Ученик повторяет основы (компоненты, замыкание цепи, мастер-промпт) — но с новым проектом. Это закрепляет навык без зубрёжки.

| | M1 | M2 | M3 |
|---|---|---|---|
| Сигнал | Цифровой | Цифровой | ШИМ |
| Новое | Основы + первый код | Датчик + `if/else` | PWM, аналоговое управление |
| Повторяем | — | Всё из M1 | Всё из M1+M2 (кроме реле) |
| Железо | LED, кнопка, реле, Pico | + ИК-датчик, вентилятор 12V | LED + кнопка, без реле |

## Общие принципы Этапа 1

1. **Ученик работает САМ** по методичке. Ментор — разблокировщик, не лектор
2. **ИИ с первого модуля** + системный Socratic-промпт
3. **3+ точки выбора** в каждом модуле (тип мышления виден)
4. **Gradual Release** — поддержка убывает от M1 к M3
5. **Портфолио** — каждый модуль завершается фото+видео+кодом в личной странице ученика

## Структура каждого модуля

Любой модуль Этапа 1 = 6 этапов по методике **6D**:

```
📋 DEFINE → 🎨 DESIGN → 📐 DRAFT → 🔧 DETAIL → 🚀 DEPLOY → 🎬 DEMO
```

Внутри каждой фазы — темы. В каждой теме цикл: **🛠 Практика → 💡 Теория → 🧩 Выводы → ✅ Чекпоинт**. После каждой фазы — **🛑 Стоп. Проверьте себя** (метакогнитивные вопросы без подглядывания).

## Чего нет в Этапе 1 (намеренно)

- Нет моторов
- Нет связи (Wi-Fi/Bluetooth)
- Нет сложной логики (state machines, interrupts)
- Нет платы расширения без клемм (паяльник не трогаем)

Всё это — в Этапах 2 и 3.
