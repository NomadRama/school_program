# Диаграммы Модуля 1

SVG-иллюстрации модуля. Каждая — самодостаточный файл, открывается в браузере или редактируется в любом редакторе SVG (Inkscape, Figma, текстовом редакторе).

## Как обновлять

1. Редактировать SVG напрямую в редакторе (Inkscape, VS Code)
2. Или переделать в HTML (`docs/module-1/index.html`) → запустить `python3 tools/extract_svgs.py` → перезапишет эти файлы

## Стиль

Все диаграммы в едином стиле:
- Белый фон компонентов, скруглённые углы, тень
- Цвета линий-проводов:
  - 🔴 красный `#C94C4C` — сигнал (GPIO от Pico)
  - 🟠 оранжевый `#E8A33D` — питание
  - ⚫ чёрный `#2B2B2B` — GND (земля)
  - 🟡 жёлтый `#F0C040` — силовая нагрузка к LED
- Клеммы компонентов — кружки с подписями на краях прямоугольника
- Текст: sans-serif, подписи пинов — monospace
